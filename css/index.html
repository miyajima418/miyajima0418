一人ひとりに
　　　心地よさを

青森市でバーをお探しなら、当店へ。
当店は『名前の無い』bar。 名前も無ければ、ルールも、ハードルもありません。カウンターで私たちとお話する人、静かにしっかり飲む人、会社の同僚や友達と楽しく飲む人、恋人とゆったりとした時間を過ごす人。そんな誰もが、肩ひじ張らずに過ごせるような心地の良い空間をづくりをしております。

実は元教師という異色の経歴を持つ当店のマスターです。教師時代に通っていたバーの温かさに感動し、自分でもバーを開きたいという思いが強くなり、退職して、お店を作りました。

「初めて行くから緊張するなあ。」
「あんまりお酒知らないんだよなあ。」
「女性一人でも大丈夫なのかなあ。」

バー初心者の方だと、きっと緊張や不安がありますよね。でもご安心ください。バーデビューお手伝いいたします。分からないことや不安なことはお気軽にご相談ください。私たちはお客様のことを第一に考えています。誰かと話したいとき、リラックスしたいとき、ふと人生に疲れたとき、ほっと一息ついて、明日から頑張れる。そんな皆さまの癒しの場になれたら幸いです。

青森県、野辺地という田舎出身の彼女は、みんなのお姉さん的存在です。また占いに精通しており、彼女のアドバイスで、恋人ができたお客さまが多数いるんだとか。

季節を感じる１杯を

旬の果物を使ったフレッシュフルーツカクテルは、彩りも華やかで男女問わず、とても人気です。その季節の香りや味わいを凝縮させた一杯には気分も上がります。 甘口や少し酸味のきいたもの、アルコールが強めや弱めなど、お客様の好みや気分をお聞かせください。

まるごと・いちご

いちごの酸味と甘味が楽しめるいちご好きにはたまらない１杯です。いちごは、甘味と酸味のバランスがとても良く、香り豊かで、カクテルに合う、「サマールビー」を使用。青森の農家様より直接、仕入れております。度数も低めですので、お酒が強くないお客様でも飲みやすくなっております。
